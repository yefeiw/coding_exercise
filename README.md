### Miniprojects

Small, focused projects showcasing specific programming concepts and practical implementations:

#### Linux File Path Simulation
A robust implementation of Linux `cd` command functionality with advanced features:
- **Advanced Path Resolution**: Handles complex path operations (`.`, `..`, multiple slashes)
- **Tilde Expansion**: Supports `~` notation for home directory reference
- **Symbolic Link Management**: Intelligent resolution of nested and overlapping symlinks
- **Shell-like Interface**: Enhanced wrapper class with persistent state management
- **Fluent API**: Method chaining for intuitive command sequences

#### Financial Account Manager
A comprehensive financial account balance management system with time-based tracking:
- **Grant Management**: Supports adding funds with expiration timestamps
- **Spend Tracking**: Records all expenditures with transaction timestamps
- **Efficient Calculation**: Uses binary search for optimal balance computation
- **Sorted Data Structures**: Maintains grants sorted by expiration and spends sorted by timestamp
- **Validation**: Ensures proper financial transaction integrity

#### In-Memory Key-Value Database
A thread-safe, in-memory database with comprehensive transaction support:
- **Core Operations**: GET, SET, DELETE, and COUNT functionality
- **Transaction Support**: BEGIN, COMMIT, ROLLBACK operations with nested transaction handling
- **Reverse Indexing**: Efficient value counting using reverse index mapping
- **Thread Safety**: Implements read-write locks for concurrent access
- **In-Memory Storage**: Fast data access with transaction isolation guarantees
